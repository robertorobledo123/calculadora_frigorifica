# Limitaciones

- La aplicación entrega cálculos preliminares, no una selección final.
- No sustituye normas, memoria firmada, revisión de seguridad ni software del fabricante.
- No calcula propiedades termodinámicas reales de CO2, R717 u otros refrigerantes.
- No dimensiona tuberías, válvulas, recipientes, controles, aceite ni seguridad.
- No verifica capacidad real de evaporadores o compresores a partir de catálogos.
- Los factores de protección de puertas son configurables y estimados; no son universales.
- Las propiedades de productos sin fuente declarada son preliminares.
- La infiltración por puerta es una estimación; para tráfico intenso debe contrastarse con mediciones o metodología de ingeniería más detallada.
- Los consumos eléctricos dependen del COP o datos de desempeño ingresados.
- GitHub Pages sirve una aplicación estática: no hay backend ni validación externa de catálogos.
